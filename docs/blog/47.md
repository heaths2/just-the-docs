## CLI NAT 설정

```bash
config firewall vip
edit mysql
set comment mysql
set extintf any
set extip 192.168.250.1
set mappedip 10.1.255.200
set portforward enable
set protocol tcp
set extport 3306
set mappedport 3306
end
```
