## 개요

> - tar usage
{: .new }

### tar Compress

```bash
tar -cf archive.tar foo bar 
OR
tar -zcf - foo bar | (pv -p --timer --rate --bytes > sample.tgz)
OR
tar -Jcf - foo bar | (pv -p --timer --rate --bytes > sample.txz)
```

### tar Decompress

```bash
tar -xf archive.tar
OR
pv sample.tgz | tar -zxf - -C sample/
OR
pv sample.txz | tar -Jxf - -C sample/
```
