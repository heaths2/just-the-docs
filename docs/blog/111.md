## 개요

> - Cisco Switch SNMP 설정
{: .new }

### ACL 설정
  
```bash
switch# configure terminal
switch(config)# ip access-list standard 1
switch(config-std-nacl)# permit 192.168.1.1
switch(config-std-nacl)# exit
```

### SNMP 설정

```bash
switch(config)# snmp-server community public RO 1
switch(config)# end
switch# write memory
```
