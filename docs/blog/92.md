## 개요

> - `Wget` 사용법
{: .new }

### 특정 확장자 전체 다운로드

```bash
wget -c -r --reject-regex=".*(a-c).tar.gz" -A "d*.tar.gz" -nd -P /DATA http://192.168.56.110:8000/
```

### 특정 파일 다운로드

```bash
wget -c -r -P /DATA http://192.168.56.110:8000/ansible.tar.gz
```
