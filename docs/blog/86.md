## 개요

> - IPMI(Intelligent Platform Management Interface)
{: .new }

> - sdr(Sensor Data Repository)
> - sel(System Event Log)
{: .important }

### ipmi 시스템 이벤트 로그 확인

```bash
ipmitool sel list
```

### ipmi Type 확인

```bash
ipmitool sdr type
```

### Fan 상태 확인

```bash
ipmitool sdr type 0x04
# 또는
ipmitool sdr type "Fan"
```

### Power Supply 상태 확인

```bash
ipmitool sdr type 0x08
# 또는
ipmitool sdr type "Power Supply"
```
