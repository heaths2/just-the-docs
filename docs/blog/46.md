객체 생성

```bash
config firewall address
edit Infra_0000001
set type ipmask
set subnet 10.1.255.224 255.255.255.255
or
set subnet 10.1.255.224/32
set comment Infra_0000001
end
```

IP 범위 지정 객체 생성

```bash
config firewall address
edit InfraTeam
set type iprange
set start-ip 10.1.255.224
set end-ip 10.1.255.239
set comment InfraTeam
end
```

### 방화벽 정책 설정

```bash
config firewall policy
edit 2
set name InBound
set srcintf wan2
set dstintf internal
set srcaddr all
set dstaddr InfraTeam
set schedule always
set service ALL
set action accept
set nat enable
set logtraffic utm
end
```
