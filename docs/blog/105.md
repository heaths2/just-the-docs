## 개요

> - LVM 디스크 확장/축소
{: .new }

### LVM 디스크 확장

- 논리 볼륨 확장

```bash
lvextend -L +1G /dev/Disks/domain-data
```

- 파일 시스템 확장

```bash
resize2fs /dev/Disks/domain-data 1G
```

### LVM 디스크 축소

- 파일 시스템 축소

```bash
resize2fs /dev/Disks/domain-data 1G
```

```bash
lvreduce -L -1G /dev/Disks/domain-data
```
