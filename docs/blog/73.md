### SSH Keygen 생성

```bash
ssh-keygen -t rsa -b 4096 -N "" -f /root/.ssh/id_rsa
```


### SSH Public Key 복사

```bash
ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.56.103
```
