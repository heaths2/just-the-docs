### CPU Thread Core 확인

```bash
dmidecode -t 4 | grep -E 'Core Count|Thread Count'
```

```bash
dmidecode -t processor | grep -E 'Core Count|Thread Count'
```

```bash
echo "Thread Count:" $(grep -c processor /proc/cpuinfo)
```
