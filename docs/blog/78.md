## 개요

> - yaml 파일 Route 설정
{: .new }

### 현재 세션 적용

```bash
ip route add 192.168.56.0/24 dev eth0 proto kernel scope link src 192.168.56.110
```

### permanent 적용

```bash
vi /etc/netplan/00-installer-config.yaml

network:
  version: 2
  renderer: networkd
  ethernets:
    eth0:
      dhcp4: true
      optional: true # systemctl disable systemd-networkd-wait-online.service
    eth1:
      dhcp4: false
      optional: true # systemctl disable systemd-networkd-wait-online.service
      addresses: [ 192.168.56.110/32 ]
      nameservers:
        addresses: [ 8.8.8.8 ]
      routes:
        - to: default
          via: 192.168.56.255
          metric: 200
          on-link: true
        - to: 192.168.56.0/24
          from: 192.168.56.110
          scope: link
          metric: 200
```
