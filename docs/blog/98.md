## 개요

> - xcp-ng xensource-inventory 환경설정
{: .new }

### xcp-ng

```bash
# VCPU 설정
sed -i "s/DOM0_VCPUS='.*'/DOM0_VCPUS='8'/" /etc/xensource-inventory
```

```bash
# MAX Memory 및 Memory 설정
sed -i "s/DOM0_MEM='.*'/DOM0_MEM='16384'/" /etc/xensource-inventory
```
